# AI싱크클럽 & 계산기

이 프로젝트는 두 개의 웹 애플리케이션을 포함합니다:

## 🎯 프로젝트 구성

### 1. AI싱크클럽 랜딩 페이지
- **파일:** `index.html`
- **설명:** AI Think Club 커뮤니티 소개 페이지
- **특징:** Liner 스타일의 보라색 그라디언트 디자인

### 2. 계산기
- **파일:** `calculator.html`
- **설명:** 웹 기반 계산기 애플리케이션
- **특징:** 키보드 입력 지원, 모던한 UI

## 🚀 실행 방법

```bash
# 랜딩 페이지 열기
open index.html

# 계산기 열기
open calculator.html
```
